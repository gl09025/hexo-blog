最简短的数组去重

es5
```javascript
[1,2,3,1,'a',1,'a'].filter(function(ele,index,array){
    return index===array.indexOf(ele)
})
```

es6
```javascript
[1,2,3,1,'a',1,'a'].filter((ele,index,array) => index===array.indexOf(ele))
```

es6
```javascript
[...new Set([1,2,3,1,'a',1,'a'])]
```
