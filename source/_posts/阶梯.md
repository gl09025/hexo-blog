>n 级楼梯 每次可以下 1级或者2级  问 n=20 那么一共有多少种下法?


```javascript
function down(n) {
    if (n === 1) {return 1}
    if (n === 2) {return 2}
    return down(n-1)+down(n-2)
}
```
